# GAMES — アーケードゲーム・カタログ

全ゲーム共通方針:
- React + **Canvas2D** + `requestAnimationFrame`（固定タイムステップ/accumulator）。`setInterval`・DOMグリッドは不可。
- ゲーム状態は `useRef`、React state は phase(`ready/playing/over`) と表示score のみ（毎tick setStateしない）。
- 共通土台 `app/sansu-100/lib/minigame-core.ts` に乗せる。純粋ロジックは `lib/games/<id>.ts`（テスト対象、React非依存）。
- 入力はキーボード ＋ タブレット用オンスクリーン操作の両対応。
- **コインを使って遊ぶ**（参加費 PLAY_COST / コンティニュー）。**報酬は限定バッジのみ**（コインは増やさない）。
- 新ゲーム追加 = ここに1エントリ書く → `BACKLOG.md` に周A〜Dの4タスクを積む。

各エントリの形式: `id / 表示名 / 操作 / コア仕様 / 受入基準 / 報酬バッジ閾値`

---

## 1. snake — スネーク（障害物避け）★基準実装・第1弾
- 操作: 矢印/WASD ＋ オンスクリーン十字。進行方向に自動で進む。
- コア仕様: グリッド20×20。food を取ると体が伸びスコア+1。壁/自分の体/障害物に当たると game over。
  難易度は時間経過で速度漸増、障害物（固定 or 漸増する壁）。
- 受入基準: クラッシュ無し / food取得で成長 / 衝突で over / 再開可 / 参加費が引かれ報酬でコイン増えない / 軽い。
- 報酬バッジ閾値: スコア10で「ヘビつかい🐍」、30で「ヘビの王さま👑🐍」。

## 2. runner — よけよけランナー（endless dodge）★第1弾推奨
- 操作: スペース/上 でジャンプ（または タップ）。1ボタン。
- コア仕様: キャラが自動で右へ進む（横スクロール）。地面を走り、迫る障害物をジャンプでよける。
  当たると over。進んだ距離=スコア。恐竜ゲーム風。速度漸増。
- 受入基準: ジャンプが効く / 障害物判定が正確 / 距離でスコア加算 / 再開可 / コイン消費正・報酬でコイン増えない / 軽い。
- 報酬バッジ閾値: 距離500で「かけっこ名人🏃」、1500で「風のランナー🌪️」。

## 3. whack — もぐらたたき（whack-a-mole）
- 操作: クリック/タップ。
- コア仕様: 3×3 等の穴からモグラがランダムに出る。制限時間(例30秒)内にタップして得点。たまに叩いてはいけない的（ばつ）。
- 受入基準: 出現/タップ判定が正確 / 時間切れで終了 / NG的でペナルティ(得点減のみ、コイン没収はしない) / 軽い。
- 報酬バッジ閾値: 20匹で「もぐらマスター🔨」。

## 4. breakout — ブロック崩し（簡易 breakout）
- 操作: 左右移動（矢印 ＋ オンスクリーン）でパドル操作。
- コア仕様: ボールをパドルで跳ね返し上部ブロックを崩す。全消し or ボール落下まで。反射の物理は簡易（角度反転）。
- 受入基準: 反射が破綻しない / ブロック消去でスコア / ボール落下で over / 再開可 / 軽い。
- 報酬バッジ閾値: 1面クリアで「ブロックくずし職人🧱」。

## 5. falling — おちものよけ（falling dodge）
- 操作: 左右移動。
- コア仕様: 上から物が降ってくる。左右に動いてよける。生存時間=スコア。落下数/速度が漸増。
- 受入基準: 当たり判定が正確 / 生存時間でスコア / 再開可 / 軽い。
- 報酬バッジ閾値: 30秒生存で「よけ名人💨」。

## 6. memory — メモリーめくり（神経衰弱）★アクション苦手な子向け
- 操作: クリック/タップ。時間制限なし（or 緩め）。
- コア仕様: 裏返ったカードを2枚めくり、絵柄が揃えば取得。全ペア取得でクリア。手数が少ないほど高評価。
- 受入基準: めくり/一致判定が正確 / 全消しでクリア / 手数カウント / 軽い。
- 報酬バッジ閾値: 規定手数以内クリアで「きおくの達人🧠」。

## 7. maze — めいろ（maze runner）
- 操作: 上下左右移動。
- コア仕様: 自動生成 or 固定の迷路をスタートからゴールへ。制限時間 or 最短手数で評価。
- 受入基準: 壁衝突しない / ゴール判定 / タイム計測 / 軽い。
- 報酬バッジ閾値: 制限時間内ゴールで「めいろ探検家🗺️」。

## 8. flappy — ぱたぱた（フラッピー系・追加）
- 操作: タップ/スペースで上昇（1ボタン）。重力で落下。
- コア仕様: アバターが鳥になり、すき間のあるパイプをくぐる。くぐった数=スコア。天井/床/パイプで over。子ども向けにすき間広め。
- 受入基準: タップで上昇 / すき間で加点 / 衝突で over / 再開可 / コイン消費正・報酬でコイン増えない / 軽い。
- 報酬バッジ閾値: 10で「ぱたぱた名人🐤」、30で「ぱたぱた王👑」。
